# Stock Recommendation using MCP

This code is used in a blog post [here](https://medium.com/@ngyibin/stock-recommendation-using-anthropic-mcp-b5aca90165a7?sk=f2de92fcbb915232a3d630f58ea6de7d).

To run the project:
```
uv init
uv venv
source .venv/bin/activate
pip install -r requirements.txt
uv run stock_mcp_client_prompt_resource.py
```
